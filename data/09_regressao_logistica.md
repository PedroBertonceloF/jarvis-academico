# Regressão Logística

Regressão logística é um modelo supervisionado usado principalmente para classificação binária. Ela usa a função logística, também chamada de sigmoide, para transformar uma combinação linear dos atributos em um valor entre 0 e 1.

A saída pode ser interpretada como probabilidade de pertencer à classe positiva. Se a probabilidade for maior que um limiar, normalmente 0,5, o exemplo é classificado como positivo; caso contrário, como negativo.

A função sigmoide é importante porque limita a saída do modelo e permite interpretar o resultado como probabilidade. O treinamento ajusta os pesos para reduzir o erro entre as probabilidades previstas e as classes reais.
